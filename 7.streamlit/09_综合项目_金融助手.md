# 09 综合项目：金融助手

## 项目目标

构建一个可演示的 Streamlit 应用，包含：

1. 实时行情看板
2. 舆论看板
3. 多轮聊天助手

## 项目文件

- 入口：`src/09_finance_assistant_app.py`
- 页面：`src/pages/*.py`
- 公共数据：`src/lib/mock_data.py`

## 功能拆解

1. 行情页：指标卡 + 折线图 + 明细表
2. 舆论页：情绪条形图 + 热度散点图
3. 聊天页：`st.chat_message` + `session_state`

## 可扩展方向

1. 接入真实行情 API
2. 接入向量检索 + LLM
3. 增加用户级权限与会话持久化
