## 协程
### 1 基本概念
协程（`Coroutine`）可以理解为“用户态可暂停/恢复的函数”，本质是**一种更轻量的并发单位**。

>核心特点

- 协程不是操作系统线程，调度通常由程序自己控制（如事件循环）。
- 一个线程里可以有很多协程。
- 协程在遇到 `await` 时主动让出执行权，不是被系统强制抢占。
- 很适合 I/O 密集场景：网络请求、数据库访问、文件读写。
- 切换开销通常比线程更小，但不适合纯 CPU 密集计算。

>和进程/线程对比
- 进程：隔离强、最重。  
- 线程：系统级并发、较轻。  
- 协程：用户态并发、最轻（通常）。

### 2 基本使用
>Example: 定义一个协程
```py
async def func():

    return
```

>Example: 执行一个协程
```py
async def func(duration):
    asyncio.sleep(duration)
    return "dummy"

# 方式一：asyncio.run
result = asyncio.run(func(duration=2))

# 方式二：await
async def main():
    result = await func(duration=2) # 阻塞等待结果
    return

# 方式三：asyncio.create_task

```
