## 协程

协程（`Coroutine`）可以理解为“用户态可暂停/恢复的函数”，本质是**一种更轻量的并发单位**。

核心特点：

- 协程不是操作系统线程，调度通常由程序自己控制（如事件循环）。
- 一个线程里可以有很多协程。
- 协程在遇到 `await` 时主动让出执行权，不是被系统强制抢占。
- 很适合 I/O 密集场景：网络请求、数据库访问、文件读写。
- 切换开销通常比线程更小，但不适合纯 CPU 密集计算。

和进程/线程对比一句话：

- 进程：隔离强、最重。  
- 线程：系统级并发、较轻。  
- 协程：用户态并发、最轻（通常）。

在 Python 里常见就是 `asyncio`：

- 用 `async def` 定义协程函数
- 用 `await` 挂起等待
- 用事件循环调度多个协程并发执行
