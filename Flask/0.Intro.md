# Flask
Flask 是一个轻量级的 Python Web 框架，基于 Werkzeug WSGI 工具包和 Jinja2 模板引擎。它的设计哲学是 "micro"（微框架），即它仅提供 Web 开发的基本功能，而其他功能可以通过扩展实现。

## 1 Flask 的特点
- `轻量级`：Flask 本身只包含最基础的功能，如路由和请求处理，其他功能通过扩展提供
- `灵活性`：不像 Django 这样的全栈框架，Flask 允许开发者自由选择数据库、模板引擎、身份认证方式等
- `基于 WSGI`：Flask 遵循 WSGI（Web Server Gateway Interface）标准，支持多个 WSGI 服务器，如 Gunicorn、uWSGI。
- `Jinja2 模板引擎`：用于生成动态 HTML 页面，支持模板继承、过滤器等功能。
- `丰富的扩展生态`：Flask-RESTful、Flask-SQLAlchemy、Flask-Login、Flask-WTF 等扩展可用于快速开发 Web 应用

## 2 Flask 常用功能和示例
### 2.1 安装 Flask
```py
pip install flask
```
### 2.2 创建最简单的 Flask 应用
```py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

if __name__ == '__main__':
    app.run(debug=True)
```
- `Flask(__name__)`：创建 Flask 实例，`__name__` 让 Flask 知道应用所在的模块
- `@app.route('/')`：定义路由，即访问 `/` 时执行 `home()` 函数
- `app.run(debug=True)`：启用调试模式，代码更改时自动重启，并显示详细错误信息


## 3 详细解析
### 3.1 `/` 代表 Web 应用的根路径
`/` 是 URL 结构中最顶层的路径，相当于 网站首页
例如：
- `http://example.com/` → 访问 `/` 路由
- `http://example.com/about` → 访问 `/about` 路由

### 3.2 路由（Route）定义

Flask 使用 `@app.route()` 装饰器来定义 URL 路由

```py
@app.route('/')
def home():
    return "Welcome to the homepage!"
```
- 访问 / 时，浏览器会显示 Welcome to the homepage!

### 3.3 路径层级
- `/about` 代表 `http://yourdomain.com/about`
- `/user/<username>` 代表 `http://yourdomain.com/user/jack`，其中 `<username>` 是一个变量。
- `/products/shoes` 代表 `http://yourdomain.com/products/shoes`
